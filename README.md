# Jose Hernandez LLC

Lead-stage mockup or proposal site for a handyman-services business.

## Role In The Business

- This repo is a sales asset, not a main maintained production surface right now.
- It helps CSolutions show a lead what their site could look like without touching the main business site.
- Before reusing it for another lead, replace any stale lead data and contact details.

## Shared Docs

- `AGENTS.md`
- `CLAUDE.md`
- `AI-WORKFLOW.md`
- `SECURITY-CHECKLIST.md`

## Stack

- Next.js Pages Router
- React 18
- Vercel-ready deployment
- `translations/` for i18n support

## Local Development

```bash
npm install
npm run dev
```

Useful build commands:

```bash
npm run build
npm run start
```

## Main Editing Surfaces

- `pages/` for route content
- `components/` for shared sections
- `translations/` for language content
- `public/` for real images when the mockup moves closer to delivery

## Working Rules

- Treat this repo like a reusable proposal template.
- Do not give it the same weekly priority as `Portfolio/` or a live client site unless it becomes active work.
- Remove or replace real lead data before showing it to the next prospect.

## Security Notes

Run `SECURITY-CHECKLIST.md` before demos or re-use. For mockup repos, stale contact data and copied form endpoints are usually the first things to clean up.
